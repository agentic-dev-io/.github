## Focus Areas

- **Agent Frameworks** — Multi-agent orchestration, planning, memory, and tool use
- **MCP Ecosystem** — Model Context Protocol servers, bridges, and integrations
- **Claude Code Plugins** — Skills, commands, and connectors for knowledge workers
- **DuckDB** — Extensions, CMS, and analytical backends for agent systems
- **Godot Tooling** — AI-powered game editor control via MCP and CLI
